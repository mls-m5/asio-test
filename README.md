Asio network tests
==================

Some short programs to test to implement networking with asio.

## How to build.
Download and install [matmake](https://github.com/mls-m5/matmake).

run 

```sh
matmake
```

## How to run
Open two terminals.

### Tcp
```sh
./tcpserver
```

and in another window:

```sh
./tcpclient
```


### Udp
```sh
./udpserver
```

and in another window:

```
./udpclient
```
